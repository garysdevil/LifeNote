# 槟榔
- 槟榔制品的主要分类其实就四种
    1. 榔青：新鲜的槟榔，通常用于直接嚼食，口感清脆，含有丰富的水分和营养成分。
    2. 榔咸：盐渍槟榔，经过腌制处理，口味较咸，常见于一些地方的传统消费习惯。
    3. 榔干：干槟榔，经过干燥处理，便于保存，通常用于长时间的嚼食。
    4. 榔玉：槟榔的种子部分，通常用于药用，具有一定的药理作用，常见于中药配方中。

1. 榔青，也就是所谓的新鲜生槟榔，这个玩意儿，只有海南、台湾等几个省吃，包个荖叶，撒点生石灰，槟榔西施卖的就是这个玩意。
2. 榔咸，以前主要是广东人吃，尤其是广州和肇庆一代，目前流传不广。所谓盐渍槟榔是将新鲜槟榔经过盐腌制处理后制成的，通常呈现出白色的外观，常搭配蒌叶和蚬灰等咀嚼，形成独特的风味组合。
3. 榔玉，是炮制过的槟榔种子，中药里面经常使用，助消化、驱虫、利水。像成药里的槟榔四消丸、四磨汤、舒肝健胃丸等。
4. 榔干，就是我们常吃的包装槟榔。


- 榔干，又可以大体上分三种
    1. 第一种，轻度加工，鲜果微烘晒干，叫做青果槟榔，以和成天下、口味王的大部分产品为代表，口感略显清甜，嚼起来比较软；
    2. 第二种，重度加工，槟榔鲜果为原料，经烟熏干燥工艺，一般是用橡木熏，再进行糖化发酵，然后点上卤水售卖，称之为烟果槟榔，老湘潭人就喜欢吃这个，以张新发的门店槟榔为代表，嚼起来硬，口味重；
    3. 第三种，比较特别，叫青果烟制，实际上这一类很难归类，也可以当做是轻度烟制的烟果槟榔，以胖哥槟榔为代表，之前有一个包装叫天生有范，就是典型的青果烟制。口感柔软，又带点烟果的重口味。
    - 如果用肉来做个类比，这三者就相当于鲜肉、烟熏腊肉、风干肉的区别。
