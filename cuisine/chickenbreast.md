2. 盐水浸泡法： 1000ml水 + 20g盐 + 一块鸡胸，冰箱冷藏浸泡30分钟（不超过1小时）。捞出擦干再烹饪。
3. 原理：盐水让肌肉纤维蛋白质分解，加热时收缩减少，锁住水分。

## 烹饪
1. 低温慢煮（最嫩、锁水最强）
    1. 适合健身餐、沙拉、直接吃。
    2. 做法：鸡胸肉用盐、黑胡椒、橄榄油抹匀，抽真空或用密封袋排尽空气，放入60-65°C恒温水浴锅中煮45-60分钟。
    3. 效果：肉质像鸡腿一样嫩，汁水全锁住。取出后两面各煎30秒上色更香。


2. 先煎后焖（家常最实用）
    1. 适合日常主菜、便当。
    2. 做法：
        1. 鸡胸肉横向片成两片，用刀背轻拍松肉。盐、黑胡椒、料酒腌制15分钟。
        2. 平底锅倒油，中大火每面煎1.5分钟至金黄。
        3. 转最小火，倒入小半碗水（或白葡萄酒），立刻盖盖子焖5-6分钟。
        4. 关火再焖2分钟。出锅后切片。
    3. 效果：外焦里嫩，内部有肉汁。

3. 水滑/低温水煮（极嫩、无油）
    1. 适合减脂餐、凉拌鸡丝。
    2. 做法：鸡胸肉冷水下锅，加姜片、料酒。水刚冒小泡（约80°C）就转微火，保持水面不沸腾，煮10-12分钟。关火再焖5分钟。
    3. 效果：撕成丝拌黄瓜、辣椒油，不柴不塞牙。


4. 烤（香口、省事）
    1. 适合烤箱或空气炸锅。
    2. 空气炸锅：180°C，先烤6分钟，翻面再烤5-7分钟。
    3. 烤箱：200°C，15-20分钟。
    4. 关键：用酸奶或蛋黄酱+香料腌制2小时以上，可大大增加嫩度。
