## 气血/阴阳 ——  电力系统模型
### 系统基础元件（物质层）
- **气 = 发电机组**
  核心功能是**产生能量**，代表人体一切功能活动的原动力。
- **血 = 输电网络 + 储能电池**
  既是能量输送的通道，也是储存能量的物质载体。

### 系统运行法则（关系层）

- **阳 = 发电模式 + 负载运行**
  体现为**能量的释放与转化过程**，如电流做功产生光/热/动力。
- **阴 = 储能模式 + 电路基础**
  体现为**能量的储存与承载状态**，如电池蓄电、导线绝缘层。

###  动态关系精解
#### **▶ 健康状态（阴阳平衡）**

```text
白天（阳主时段）：  
发电量(气) = 用电量 + 储电量  
→ 人体：精神充沛、体温正常、代谢旺盛  

夜晚（阴主时段）：  
用电量 ≤ 储电量(血)释放  
→ 人体：安静入睡、体温略降、修复储能
```

#### **▶ 病理状态举例**

| **故障类型** | **电力系统表现**                           | **人体对应症状**           |
| :----------- | :----------------------------------------- | :------------------------- |
| **气虚**     | 发电机老化，最大输出功率下降               | 乏力、气短、易感冒         |
| **血虚**     | 电网容量不足，电池老化                     | 面色苍白、头晕、月经量少   |
| **阳虚**     | 发电量不足 **且** 输出电压过低（产热不足） | 畏寒肢冷、喜热饮、小便清长 |
| **阴虚**     | 电池严重亏空 **且** 电路绝缘不良（虚热）   | 五心烦热、口干舌燥、盗汗   |
| **气滞**     | 输电线路局部拥堵                           | 胀痛、情绪抑郁、喜欢叹气   |
| **血瘀**     | 线路锈蚀/局部短路                          | 刺痛、舌有瘀斑、月经血块   |

1. 气虚： 气短乏力神疲；精神力气差，容易累困，人慵懒，爱睡觉，萎靡不振，缺乏活力。
2. 血虚： 面白指甲白眼睑白舌白唇白脉细；血少，在女子表现明显的一个特征就是月经量少，另个表现可以看是不是很瘦，瘦子则血虚，血虚就是血少，瘦，自然就少。

## 气血

1. 五脏六腑 
    1. 肝、心、脾、肺、肾
    2. 胆、小肠、胃、大肠、膀胱、三焦

2. 气血是五脏六腑加工饮食水谷化生而成的。
    1. 饮食水谷： 这是气血原料，每天吃进去的东西。
    2. 五脏六腑： 脏腑分工协作，对饮食水谷进行各种加工代谢之类，进而化生气血。

3. 气血常见五个消耗途径
    1. 眼睛；眼睛得血能视，肝靠血滋养，用眼过度导致必伤到肝。
    2. 心；思虑过度。
    3. 熬夜；透支气血、精气神。
    4. 纵欲；精血同源，肝肾同源，可以说纵欲过度泄精，首先伤害的是肾，逐步的会伤害肝，脾，肺，心，这些东西受到伤害，功能衰弱，气血的生成就出问题。
    5. 外邪；通俗点就是潜在疾病，慢性病，比如溃疡，炎症，肿瘤等等，这些早期可能没什么明显症状，也不易觉察，但是长在身体里是无时无刻和脏腑争夺气血能量，这些外邪成长发展也需要气血支持。

## 肌肉劳损
### 中医解释的三种原因
1. 外邪侵袭，闭阻经络（最常见诱因）：
    1. 风寒湿邪：颈部暴露于空调、风扇、窗口冷风下，或睡眠时肩颈外露。
        1. 寒性收引：使颈部筋肉、血管痉挛，气血运行不畅，不通则痛。
        2. 湿性黏滞：导致局部沉重、酸胀，症状缠绵难愈。
        3. 风性走窜：疼痛位置可游走不定，或牵扯至肩背。

2. 慢性劳伤，气血瘀滞：
    1. 长期姿势不正：如长期歪头接电话、单肩背包、长时间低头看手机电脑（现代最主要诱因）。这会导致一侧肌肉、筋膜被持续牵拉，气血供应受阻，局部形成“气滞血瘀”的病理状态。
    2. 急性扭挫伤后遗：旧伤未彻底治愈，局部经络不通，形成“陈伤”，遇诱因极易复发。

3. 外伤筋脉，血溢脉外：
    1. 突然的撞击、落枕、不当的颈部按摩或暴力扳法，可直接损伤筋脉，导致局部出血、水肿，形成瘀血，阻滞气血运行。

### 中西医对比

| 维度               | **中医（经络气血视角）**                                     | **西医（肌肉筋膜视角）**                                     |
| :----------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **核心理念**       | **功能性、关系性网络**。强调气血能量在“经络”通道中的流通与分布状态。 | **结构性、物质性组织**。关注肌肉、筋膜、肌腱、韧带等实体组织的解剖、生物力学与病理改变。 |
| **对“劳损”的理解** | **气血瘀滞，筋失所养**。是整体气血运行不畅在局部的表现，与肝、肾、脾等内脏功能相关。 | **组织微损伤与炎症**。是肌肉/筋膜因过度或不当使用导致的微小撕裂、无菌性炎症、粘连和痉挛。 |
| **病因分析**       | 外感风寒湿邪、内伤七情、劳倦、姿势不正等导致**气血失衡**。   | 重复性压力、急性损伤、不良姿势、核心肌群无力等导致**生物力学失衡**。 |
| **治疗重点**       | **恢复“流通”与“平衡”**。疏通经络，调和气血，补益脏腑。       | **修复“结构”与“功能”**。消除炎症，松解粘连，强化肌肉，纠正力学。 |


## 中医治疗
1. 内治法（通过体内吸收）
    1. 中药：汤、丸、散、膏、丹
2. 外治法（通过体表刺激或吸收）
    1. **针法**：**针灸**（毫针、电针、耳针等）
    2. **灸法**：**艾灸**（直接灸、隔物灸、艾条灸）
    3. **手法**：**推拿、正骨**
    4. **物理法**：**拔罐、刮痧、放血**
    5. **自然疗法**：**导引（太极拳/八段锦）、饮食、情志**
    6. **其他**：**贴敷、熏洗**等
